⚓✨ Nautical x Magical x Luxury | Swag Strategy Guide (v2)

Event: 4-Night / 5-Day Bahamas Cruise
Audience: Loyal repeat guests + influencers
Vibe: Yacht club + beach party + “magic” polish
Swag rule: Fewer pieces, higher quality, used daily on the ship + kept at home later

🎌 Visual Worldbuilding (Flags, Banners, Signage) — “Make it Feel Like a Branded Voyage”

These are high-impact and don’t create post-event clutter like cheap giveaways.

Simple but powerful concept: Create one “event crest” (yacht club badge) + 2–3 supporting icons. Use it everywhere.

✅ Suggested “Premium Core Set” (Fewer, Better)

If you want a clean, premium set that hits all vibes:

Stateroom Lux (choose 2–3): robe + slippers + blanket (and optional spray)
Beach: premium towel + wet bag + cooler bag + coozies
Utility: slim power bank + luggage scale
Content: selfie stick/tripod combo
Keepsake: limited edition pin

That’s a tight set that feels expensive without being clutter.
